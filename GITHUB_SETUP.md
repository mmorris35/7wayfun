## Labels to Create

Navigate to: https://github.com/mmorris35/7wayfun/labels

### Component Labels
- `firmware` - #d73a4a (red) - Firmware/CircuitPython code changes
- `simulator` - #0075ca (blue) - Desktop simulator changes
- `hardware` - #fbca04 (yellow) - Physical hardware related
- `mobile` - #1d76db (dark blue) - Mobile app features

### Type Labels (use existing when possible)
- `enhancement` - (should exist) - New features
- `bug` - (should exist) - Something isn't working
- `documentation` - (should exist) - Documentation improvements

### Process Labels
- `testing` - #5319e7 (purple) - Test infrastructure and coverage
- `breaking-change` - #b60205 (dark red) - Breaking API or behavior change
- `good-first-issue` - #7057ff (violet) - Good for newcomers
- `help-wanted` - #008672 (teal) - Extra attention needed
- `future` - #e4e669 (light yellow) - Long-term future feature
